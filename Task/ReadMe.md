# 情報可視化論 最終課題システム (Pokemon Visualization System)

本システムは、大学院講義「情報可視化」の最終課題として開発された、Webベースのインタラクティブな可視化ツールです。D3.js (v7) を使用して、ポケモンデータセットの探索的分析を実現しています。

## 1. 概要
*   **データ**: `Pokemon.csv` (ポケモン 800匹のステータスデータ)
*   **技術**: HTML5, CSS3, Vanilla JavaScript, D3.js (v7)
*   **主な機能**:
    *   **散布図 (Overview)**: 任意の2つのステータス（攻撃、防御、素早さなど）を軸として選択・比較可能。
    *   **棒グラフ (Detail)**: マウスオーバーしたポケモンの詳細ステータスを表示。
    *   **フィルタリング**: タイプごとの表示/非表示を凡例クリックで切り替え。
    *   **連携 (Linking)**: 散布図と棒グラフが連動して情報を表示。

## 2. 実行方法
本システムは `d3.csv` を使用して外部ファイルを読み込むため、ローカルサーバー経由で実行する必要があります（ブラウザのセキュリティ制限/CORS回避のため）。

### 手順
1.  **ターミナルを開く**
    ターミナルアプリを開き、本ディレクトリ (`Task`) に移動します。
    ```bash
    cd /Users/shuta800/Work/InfoVis2025/Task
    ```

2.  **ローカルサーバーの起動**
    Pythonがインストールされている場合、以下のコマンドで簡易サーバーを立ち上げます。
    ```bash
    # Python 3系
    python3 -m http.server
    ```
    ※ 実行後、`Serving HTTP on 0.0.0.0 port 8000 ...` のようなメッセージが表示されれば成功です。

3.  **ブラウザでアクセス**
    Webブラウザ（Chrome, Firefox, Safari等）を開き、以下のアドレスにアクセスしてください。
    
    [http://localhost:8000/index.html](http://localhost:8000/index.html)

## 3. ファイル構成
*   `index.html`: Webページの構造定義。
*   `style.css`: レイアウトと装飾。
*   `main.js`: D3.jsを用いた可視化ロジック（クラスベース実装）。
*   `Pokemon.csv`: 使用するデータセット。
*   `ReadMe.md`: 本説明書。

## 4. 操作方法
1.  **軸の変更**: 画面上部の「X Axis」「Y Axis」プルダウンメニューを変更すると、散布図の軸がアニメーション付きで切り替わります。
2.  **詳細表示**: 散布図上の丸（ポケモン）にマウスカーソルを合わせると、ツールチップと右側の棒グラフに詳細が表示されます。
3.  **フィルタリング**: 画面下部のタイプ別凡例（色付きタグ）をクリックすると、そのタイプの表示/非表示を切り替えられます。
